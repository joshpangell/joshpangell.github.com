Simple HTML5 history object


Inspired by JÃ¶rn Zaefferer's simple history, this is a very simple, but slightly more robust Javascript library using HTML5 history.pushstate. This library does not have support for hashes (non-html5), but stubs are created in places where this would go. There are various libraries for hash history available. Try checking out jQuery history plugin or for support of both, see History.js


Github repository


To init the history event, add `rel="history"` to any tag like 
<a rel="history" href="link-one">Link One</a>
$(document).ready(function() {
	History.Init();
});
